# **App Name**: Estadika

## Core Features:

- Start Match Button: Implement a button to start a new match.
- Tournament Mode Button: Implement a button to navigate to the tournament mode.
- Player Management Button: Implement a button to navigate to the player management screen.
- History Button: Implement a button to navigate to the match history screen.

## Style Guidelines:

- Primary color: A vibrant blue (#29ABE2) to represent energy and activity.
- Background color: Light gray (#F0F0F0) to provide a clean and neutral backdrop.
- Accent color: A vivid green (#90EE90) for interactive elements and highlights.
- Body and headline font: 'PT Sans', a humanist sans-serif, which combines a modern look and a little warmth or personality. This font is suitable for headlines or body text.
- Use simple, modern icons for the navigation buttons to enhance usability.
- Arrange the four buttons in a visually balanced and intuitive layout on the home page.
- Incorporate subtle animations on button interactions to provide visual feedback.
